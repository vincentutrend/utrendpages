# UTrend Tech Co. - 完整多頁式網站

## 📋 項目概述

這是一個完整的企業IT顧問公司多頁式網站，包含7個相互連接的頁面，採用統一的設計風格和色系。所有頁面都使用 Tailwind CSS 構建，提供現代化、響應式的用戶體驗。

## 📄 頁面清單

### 1. **index.html** - 首頁
- 英雄區段（Hero Section）
- 公司統計數據卡片
- 合作品牌展示（無限滾動輪播）
- 完整導航和頁腳

### 2. **services.html** - 服務項目
- 6個主要服務的詳細介紹卡片
  - ERP 顧問服務
  - CRM 顧問服務
  - AI 解決方案
  - 數位轉型
  - IT 基礎架構
  - 定制開發
- 行動呼籲（CTA）部分

### 3. **cases.html** - 成功案例
- 4個成功案例研究
- 每個案例包含背景、成果等詳細信息
- 成果統計數據展示

### 4. **about.html** - 關於我們
- 公司故事和背景
- 使命與願景說明
- 核心價值觀（3個主要價值）
- 選擇我們的理由

### 5. **team.html** - 顧問團隊
- 6位主要顧問的個人資料卡片
- 每位顧問的專長領域說明
- 團隊統計數據

### 6. **insights.html** - 最新觀點
- 6篇最新部落格文章/洞察
- 文章分類標籤和發布日期
- 訂閱通訊表單

### 7. **contact.html** - 聯絡我們
- 完整的聯繫表單
- 公司聯絡方式（電話、郵件、地址、營業時間）
- 響應式佈局設計

## 🔗 頁面連接

所有頁面都通過以下方式相互連接：

1. **頂部導航欄** - 包含指向所有主要頁面的連結
2. **呼籲行動按鈕** - 各頁面的 CTA 按鈕連結到相關頁面
3. **頁腳** - 完整的網站地圖和連結

導航流程：
- 首頁 ↔ 所有其他頁面
- 服務頁面 ↔ 聯絡我們
- 案例研究 ↔ 關於我們
- 團隊頁面 ↔ 所有頁面

## 🚀 如何使用

### 本地預覽
1. 將所有 HTML 檔案放在同一個文件夾中
2. 使用任何現代瀏覽器打開 `index.html` 檔案
3. 點擊導航連結在頁面間移動

### 部署到網絡
1. 將所有 HTML 檔案上傳到您的網站主機
2. 確保文件在同一目錄中
3. 設定 `index.html` 為默認頁面

### 自定義內容
編輯各 HTML 檔案中的以下部分：

```html
<!-- 公司名稱 -->
<h2>UTrend Tech Co.</h2>

<!-- 聯絡資訊 -->
<p>+886 (2) 1234-5678</p>

<!-- 社交媒體連結等 -->
```

## 🔧 進階自定義

### 修改色系
編輯 HTML 檔案中的 `<script id="tailwind-config">` 部分：

```javascript
colors: {
  "primary": "#2A52BE",      // 修改主色
  "accent": "#FF7A00",       // 修改強調色
  // ... 其他顏色
}
```

### 添加新頁面
1. 複製任何現有的 HTML 檔案
2. 修改頁面內容
3. 更新導航欄中的連結
4. 確保所有連結都指向正確的頁面

### 集成表單提交
`contact.html` 中的表單目前是靜態的。若要添加功能，需要：
1. 使用 Formspree、Netlify Forms 等服務
2. 或連接到您的後端 API

## 📝 SEO 優化建議
